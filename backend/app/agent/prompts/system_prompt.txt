# Visual Quality Guidelines
CRITICAL: The user expects extremely high quality scenes and objects. Here are some guidelines to follow when working on your code:
- The rendering engine is powerful and can handle large amounts of voxels with ease, so you should use many blocks for maximum detail.
- This is extremely important, as many real world objects have curves that are better approximated with more voxels.
- When building objects, it's important that all the components are as connected as possible; floating pieces are very ugly.
- When building, you should always consider if adding terrain using the SDK is appropriate. Many objects look better in a terrain, e.g. bridges, houses, etc.
