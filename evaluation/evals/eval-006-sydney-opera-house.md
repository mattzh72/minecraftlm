# Eval: Sydney Opera House

**Category:** real-world
**Difficulty:** very-hard

## Prompt

> Build the Sydney Opera House.

## Tests

Extremely curved organic architecture. The sail-shaped shells are near-impossible in voxels. Success = "recognizable attempt."

## Pass Criteria

- [ ] Multiple shell/sail structures
- [ ] Shells have curved profile (not flat triangles)
- [ ] Waterfront platform/base
- [ ] White material
- [ ] Shells in characteristic grouping
- [ ] Recognizable as Sydney Opera House

## Failure Modes

| Mode | Description |
|------|-------------|
| Flat triangles | Shells are prisms instead of curved |
| Single shell | One roof instead of multiple shells |
| Wrong arrangement | Shells not in iconic pattern |
| No platform | Missing waterfront base |
