# Eval: Basketball Court

**Category:** real-world
**Difficulty:** medium

## Prompt

> Generate me a basketball court.

## Tests

Symmetry and real-world accuracy. Courts have specific proportions, markings, and equipment on BOTH ends.

## Pass Criteria

**Hoops (both ends):**
- [ ] Backboard
- [ ] Rim/hoop
- [ ] Net
- [ ] Support pole

**Court:**
- [ ] 3-point line (arc) both ends
- [ ] Key/paint area both ends
- [ ] Center circle
- [ ] Boundaries (sidelines, baselines)
- [ ] **Both ends mirror each other**

**Context:**
- [ ] Court surface
- [ ] Terrain extends beyond court

## Failure Modes

| Mode | Description |
|------|-------------|
| One hoop only | Forgot the other end |
| No net | Backboard + rim but no net |
| Missing 3-point line | Court without the arc |
| Asymmetric | Ends don't match |
| Floating backboard | No support pole |
