# Eval: Pyramids of Giza

**Category:** real-world
**Difficulty:** hard

## Prompt

**Turn 1:**
> Build the Pyramids of Giza.

**Turn 2:**
> Add King Kong at the top of one of the pyramids. And add a plane flying above.

## Tests

Multi-turn scene with creative additions. Tests relative sizing, desert terrain, and 3D positioning (plane should be HIGH).

## Pass Criteria

**Turn 1:**
- [ ] 3 pyramids
- [ ] Correct relative sizes (one largest)
- [ ] Pyramid shape (tapered, pointed)
- [ ] Sand/desert terrain
- [ ] Sand-colored material

**Turn 2:**
- [ ] Recognizable gorilla figure (not a box)
- [ ] King Kong on top of pyramid
- [ ] Airplane shape (fuselage + wings)
- [ ] **Plane HIGH above scene** (not ground level)

## Failure Modes

| Mode | Description |
|------|-------------|
| Single pyramid | One instead of three |
| Lazy King Kong | Brown box instead of gorilla shape |
| Low plane | Plane at ground level |
| Same-size pyramids | All identical instead of varied |
