# Eval: Hedge Maze

**Category:** geometry
**Difficulty:** medium

## Prompt

> Create a hedge maze.

## Tests

Algorithmic thinking. Maze must be solvable with entrance/exit. Tests terrain context and creative layout (not boring grid).

## Pass Criteria

- [ ] Recognizable maze layout
- [ ] Walls made of hedge/leaf blocks
- [ ] Clear entrance
- [ ] Clear exit
- [ ] **Maze is solvable** (path from entrance to exit)
- [ ] Paths walkable (1-2 blocks wide)
- [ ] **Terrain extends beyond maze**
- [ ] Interesting layout (not simple grid)

## Failure Modes

| Mode | Description |
|------|-------------|
| Unsolvable | No path from entrance to exit |
| No entrance/exit | Fully enclosed |
| Floating | No terrain around maze |
| Boring grid | Simple grid instead of winding paths |
| Wrong material | Stone/wood instead of hedge |
