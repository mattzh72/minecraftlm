# Screenshots

Store eval run screenshots here, organized by eval ID.

## Structure

```
screenshots/
├── eval-001/
│   └── YYYY-MM-DD-{model}-{pass|fail}.png
├── eval-002/
│   └── ...
└── eval-003/
    └── ...
```

## Naming Convention

`{date}-{model}-{result}.png`

Examples:
- `2025-12-15-gemini-fail.png`
- `2025-12-15-claude-pass.png`
- `2025-12-16-gpt-fail.png`
